Prototype: one leading Core card per class in the character-sheet tent format (ancestry / loadout geometry). Replaces retired Loadout cards — folds into the Class slot on the full character sheet. Not labeled as “leading”; role is invisible at the table.

Format: Fold on the dashed line → stand upright in the Class panel of the 3-panel character sheet. Ribbon shows the card name (readable right-side-up on the lower half). Footer: ⁘ CLASS | Core ⁘. Monk + Wizard are draft concepts (no Baserow row yet).

Retired: All 9 Loadout Core cards removed from the game — chargen now flows from the Player Primer v2 steps only.
