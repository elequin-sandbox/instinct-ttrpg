PRIMARY CHANGES — DO THIS BEFORE NEXT PLAYTEST

High-confidence from debrief + Nathan’s explicit “this is what I’m doing next.”

[ ] P1 — Redefine Objectives (Player-Co-Defined Challenge Pools)

The problem: The prison break and cell fight frustrated players because objectives were implicit, shifted mid-scene, or fought the fiction (stealing keys didn’t “count” toward the pool; talking down enemies felt like it should work but didn’t map cleanly). Players expected progress the main counter; GM abstraction read as arbitrary.

The fix:

  • At scene/combat open, co-define the objective in plain language with the table (“get out of the cell,” “take the keys,” “soften him up so he gives them up”).
  • Telegraph what success looks like before rolls — including alternate paths (persuasion lane vs strike lane) when Nathan allows them.
  • Treat almost all checks (social + combat) as challenge pools: GM sets dice rows/waves; players clear them with the same tactile loop.
  • Write one primer + GM guide paragraph on objective setup; add to encounter-design checklist for Nathan.

What to preserve: Group skill check waves from the jail scene when the goal was clear — players liked knocking dice off rows.

Nathan (committed): “The number one takeaway that is actionable is redefining how objectives work… letting the players define what that is… instructing GMs to telegraph it way more.”

[ ] P2 — Crit Flourish System (Bold Keyword Self-Adjudication)

The problem: Conditions (Exposed, Rattled, Rooted, etc.) create GM adjudication load and keyword confusion. Players want crits to feel like story punctuation, not nested rules trees.

The fix — Nathan’s locked direction:

  • On crit, player checks if the card’s bold flourish word fits the current objective context; if yes, apply a printed numeric effect (no GM inventing rules mid-turn).
  • Three flourish colorings (visual on card):

- Red / offensive — remove N dice from objective pool (e.g. Exposed 2)

- Blue / defensive — help allies (Boost, redirect Threat, etc.)

- Green / resolve — heal, shed Threat, recover Resolve/Hit Die narrative

  • GM only judges fiction fit, not effect size. Optional: “remove a die of 4 or less” granularity.

Draft procedure (primer): Roll crit → read bold word → “does this apply to what we’re trying to do?” → if table agrees, apply the number.

Nathan (committed): “That’s what I’m going to do next.”

[ ] P3 — Encounter Design: Telegraph Creative Card Use in Scene 1

The problem: Joey (new player) felt disarmed outside combat — hand full of “punchy” Barbarian cards, didn’t realize he could punch a wall / stay in character mood. Veterans might improvise; new players wait for “Strike” labels.

The fix:

  • Script scene 1 (before any challenge pool) to reward one unexpected card application — e.g. Dropkick in a social scene, combat card in negotiation. GM says yes because it’s awesome.
  • Shorten scene 1 — move to fresh draw faster; “dopamine of drawing five new cards” at combat start.
  • Add one line to GM guide: “Scene 1’s job is to teach: the label is a suggestion, not a cage.”

Nathan (committed): “I failed… I should have really pushed that… make the scene shorter.”

[ ] P4 — Simplify Ability Cards (Strip Legacy Condition Complexity)

The problem: Cards still carry dense keyword baggage; reactive bumps and multi-branch crit lines weren’t yielding rewards commensurate with read cost. Smoke bomb felt unusable (player feared it would “do nothing”).

The fix:

  • Audit active pool cards for PbtA-style simplification — one clear effect + optional flourish band, not stacked forks.
  • Remove or rewrite cards that require GM to track unit-linked conditions unless P2 flourish system replaces them cleanly.
  • Playtest pared Smite / Strike templates as models.

Nathan (committed): “So many of these cards can be sipped, pared-down… by a large margin.”

SECONDARY CHANGES — NEXT SPRINT

[ ] S1 — Threat / “Trading Blows” Rhetoric + Turtling Cost

Players liked Threat once explained, but all-defend rounds felt story-stagnant (“not accomplishing anything”). Need clearer opportunity cost for ignoring the objective lane — e.g. objective rows get harder, noise meter, or enemies progress a parallel track.

Consider: Rename or subtitle Threat as incoming / trading blows; GM script when players turtle.

Parked sub-idea: Exposed on objective rows affects future waves, not just current unit HP abstraction (Gel).

[ ] S2 — Unified Social + Combat Resolution

Dan/Greg thread from prior playtests echoed here: same pool-clearing loop for jailbreak, persuasion, and brawls. Secret DCs become the exception, not the default.

Scope note: Nathan is open but wants objectives fixed first (P1).

[ ] S3 — Stealing Keys / Alternate Win Paths (Objective Branching)

When fiction shifts (pickpocket vs brawl), allow declared sub-objectives with telegraphed pools (“persuade him to hand over keys” = smaller pool) instead of punishing creative play.

Draft: Original objective remains underneath; new lane locks out old lane only when table agrees.

[ ] S4 — More Scenes Per Session (Showcase Class Cores)

Frenzy, Ace top-deck, extra vow rolls barely fired — scene count too low for core identity cards. Target 4+ scenes in future one-shots; trim scene 1 length (P3) to budget time.

[ ] S5 — Reactions: Narrative-First, Less Mechanical

Nathan still wants reactions for dynamism beyond Aid, but Playtest 3 reactions felt too mechanical. Direction: waiver/blow-out style — fiction trigger + Snap Check or simple outcome, not nested rules.

[ ] S6 — Ancestry: Deck Draw vs Snap Check

Ancestry Snap Check felt weak / situational (Dragonborn example); Nathan’s tension: always-available ancestry is “too optimal,” in-deck is simpler. Test in-deck ancestry next session with clearer fiction triggers.
