Snap Check React — Design Lab

Three React prototypes — one per Snap Check format. Setup text describes the attempt only; bands carry all outcomes. Same Miss/Crit rules as any check.

Snap Check (global rule — primer / GM guide): A one-die reaction check. Roll 1d6 — no Tier pool, no opposing pool. A natural 1 is a Miss (Toll); a natural 6 explodes (roll one bonus die and add it). Read the card's bands or face table. Cards may require 7+ for the top outcome — that only lands on a Crit. Three card formats; breakpoints are always card-specific.

Formatting rule: Everything before the Snap Check block is setup — what you try to do, not what definitely happens. Use *try to*, *attempt to*, *reach for* — never presuppose success. The bands (or face list) are the only place outcomes appear.
